1. instalirati u VS code https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint
2. u konzoli pokrenuti npm init
3. ako niste sigurni/nemate preferencije, na tim koracima pritisnite enter
4. u konzoli pokrenuti npm init @eslint/config. odabrati opcije kao u dijelu KORACI za ESLint. Odabirete polja strelicama i enterom
5. Aktivirati će se ESLint unutar VS Code i možete vidjeti grešku u kodu
   za eventualne popravke u VS Code, pritisnuti ctrl + .
